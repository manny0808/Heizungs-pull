# Brands Icons

This directory contains brand icons for Home Assistant.

## Usage

If Home Assistant's brands integration needs a custom icon for this integration,
place it here with the naming convention:

- `icon.png` - 192x192 PNG
- `logo.png` - Any size, typically for documentation

- `../custom_components/heizungs_pull/icon.png` - Main integration icon
